# Traffic Data — Get-AzPaaSAvailability

This branch stores historical GitHub traffic data and the published dashboard.

- `data/` — CSV files with daily views, clones, stars, referrers, paths, repo stats
- `index.html` — Dashboard published to GitHub Pages
- `dashboard.html` — Same as index.html (backup copy)

Data is collected by the `Collect Traffic Data` workflow on main.
